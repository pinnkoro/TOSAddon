# Other Character Skill List

> 各キャラクターの錬成スキルや強化値を一覧表示（Indun Panel の戻るボタン右クリックでも表示）

| 項目 | 内容 |
| --- | --- |
| キー | `other_character_skill_list` |
| ソース | [other_character_skill_list.lua](other_character_skill_list.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタンで一覧が開きます） |

## 使い方

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタン、または
**Indun Panel の「戻る」ボタンを右クリック**すると一覧が開きます。

1 行が 1 キャラで、左から順に

| 列 | 内容 |
| --- | --- |
| 職業アイコン | 最後の職業（Indun List Viewer に代表職の記録があればそちらを優先） |
| キャラ名 | 右クリックで**装備詳細**、左クリックで**キャラチェンジ**（Instant CC が ON のとき） |
| 武器 | RH / LH / RH_SUB / LH_SUB / RING1 / RING2 / NECK の強化値 |
| 上着・ズボン・手袋・靴 | 装備の強化値と、**付いている錬成スキルのアイコン・レベル・名前** |
| その他 | レジェンドカード / ゴッデスカード / ボルタエンブレム / アーク / レリック / イヤリング / コア |
| 右端のチェック | そのキャラを一覧から**隠す** |

キャラ名にマウスを乗せると **GearScore** が出ます。

上部のボタン:

| ボタン | 動作 |
| --- | --- |
| `?` | 並び順と操作の説明 |
| バラックアイコン | バラックへ戻る |
| チェックボックス | **「隠す」指定を有効にするか**のマスタースイッチ |

右上の ✕ か ESC キーで閉じられます。ESC で閉じるのは一番手前の 1 枚だけなので、
装備詳細を開いているときは詳細だけが先に閉じ、他の Nexus Addons P のウィンドウも開いたままです。

### 装備詳細（キャラ名を右クリック）

バラック画面と同じ見た目のパネルで、そのキャラの全装備を表示します。
**ログイン中のキャラと同じバラックのキャラだけ**対応しており、
別バラックのキャラを選ぶと

```
詳細表示は、ログイン中のキャラと同一バラックのキャラのみ対応しています
```

と表示されます。

## 記録されるタイミング

**街で** インベントリを**開いたとき／閉じたとき**に、そのキャラの装備を記録します
（0.5 秒以内の連続呼び出しは無視します）。

記録する内容:

| 部位 | 記録するもの |
| --- | --- |
| 上着 / ズボン / 手袋 / 靴 | ClassID・強化値・**錬成スキル名とレベル** |
| 武器・指輪・首飾り・コア | ClassID・強化値 |
| ボルタエンブレム | 現在のボルタエンブレムのレベル |
| アーク | `ArkLevel` |
| レリック | `Relic_LV` |
| イヤリング | **特殊オプション（職業・ランク・レベル）を全部** |
| レジェンド／ゴッデスカード | ClassID とカードレベル |
| — | GearScore（`NoRing` を除く全装備の合計） |

## 並び順

**バラック 1 → 2 → 3 の順にグループ化**し、各バラック内はバラックの並び順どおりです。

順番が正しく出ないときは、**各バラック（1・2・3）へ一度ずつログイン**してください。
そのタイミングでレイヤーと並び順が記録されます。

アカウントから消えたキャラは一覧から除去されますが、
**クライアント起動直後（バラック名簿がまだ空のとき）は削除しません**
（全キャラを誤って消さないためのガードです）。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/other_character_skill_list.lua`

```lua
return {
  ver = 1.1,
  hide = 0,
  chars = { ["<キャラ名>"] = { name = "...", cid = "...", layer = 1, order = 0,
                              hide = 0, gear_score = 0,
                              equips = { SHIRT = { clsid = 0, lv = 0,
                                                   skill_name = "...", skill_lv = 0 }, ... } } },
}
```

**JSON ではなく Lua 形式**で保存します。`other_character_skill_list.json` があれば移行します。

## 注意

* **記録は街でしか行われません。**フィールドでインベントリを開いても更新されません。
* 一覧を開く処理は `pcall` で保護されており、エラー時は
  `[Nexus Addons P] OCSL open error: ...` を表示して `debug_log.txt` に残します。
* 本家 Nexus Addons など、同名の旧初期化関数（`OTHER_CHARACTER_SKILL_LIST_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、都市入場ごとにキャラが消える不具合と、都市外から設定フレームを開けない
  不具合を修正済みです（v1.0.0）。
