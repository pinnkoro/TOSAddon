# Separate Buff Custom

> セパレートバフフレームを改造します

| 項目 | 内容 |
| --- | --- |
| キー | `separate_buff_custom` |
| ソース | [separate_buff_custom.lua](separate_buff_custom.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 何をするアドオンか

ゲーム標準の**セパレートバフフレーム**（`buff_separatedlist`＝重要バフを大きく別枠で出すあれ）に対して、

1. **好きなバフを追加で表示できる**ようにする
2. **スタック数を大きく表示**し、最大スタックに達したら色を変える（任意でエフェクトと効果音）
3. **フレームの位置を全キャラで共有する／キャラに追従させる**

の 3 つを行います。

## 使い方

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから設定画面を開きます。
ゲーム内の全バフが `<ID> : <バフ名>` の一覧で並びます（検索欄で絞り込めます）。

| 操作 | 動作 |
| --- | --- |
| 右側のチェック | **セパレートバフフレームに表示する** |
| 左側の小さいチェック | 最大スタック到達時に**エフェクトと効果音**を出す |

チェックしたバフは一覧の上の方へ並べ替えられるので、設定済みのものを見失いません。

### スタック数の表示

| 状態 | 表示 |
| --- | --- |
| 最大スタック（`OverBuff` 以上） | **赤い大きな数字**。「エフェクト表示」が ON ならこのとき 1 回だけ演出が出る |
| 2 スタック以上 | 太字の数字 |
| 1 スタック | 数字なし |

### 残り時間と点滅

* 各スロットに残り時間を表示します。
* 残り **5 秒未満**になるとスロットが点滅します。
* **プレミアムトークン**のバフは、残り **1 時間未満**になると点滅します。

### フレームの位置

設定画面の右上に 3 つのチェックボックスがあります。

| チェック | 動作 |
| --- | --- |
| 位置を各キャラ共有 | セパレートバフフレームの座標をアカウント共通で保存し、全キャラで同じ位置にする |
| 位置を固定 | ドラッグ移動を禁止する |
| 追従モード | フレームを**自キャラの頭上に追従**させる（この間は移動できません） |

**追従モードが最優先**です。追従 OFF ＋共有 ON なら共有座標、どちらも OFF ならゲーム標準の
キャラ別位置（`../release/addon_setting/buff_separatedlist/<アカウントID>/settings.json`）に戻します。

## しくみ

`BUFF_SEPARATEDLIST_CTRLSET_CREATE` / `..._REMOVE` / `..._ON_RELOAD` /
`BUFF_SEPARATED_TIME_UPDATE` をフックしています。

* 追加表示は、ゲームが「セパレート対象」と見なしていないバフ
  （`ui.buff.IsBuffSeparate(buff_id) == 0`）に対してのみ `CTRLSET_CREATE` を自前で呼ぶ形です。
  もともとセパレート対象のバフには手を出しません。
* フレームの自動配置スクリプト（`_FRAME_AUTOPOS`）は初期化時に止めます。
* OFF にすると、フックはそのままでも `INIT_BUFF_SEPARATEDLIST_UI` を呼んで標準の見た目に戻し、
  位置もゲーム標準の保存先から読み直します。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/separate_buff_custom.json`

```json
{
  "x": 0, "y": 0, "move": 1,
  "sep_buffs": { "<バフID>": { "display": 1, "with_effect": 0 } },
  "location_share": 0,
  "tracking": 0
}
```

`move` が `1` で移動可、`0` で固定です（設定画面のチェックとは逆の持ち方をしています）。
表示チェックを外したバフは、キーごと `sep_buffs` から削除されます。

## 注意

* **一覧はゲーム内の全バフを走査して作る**ため、検索せずに開くと行数が多く重くなります。
* 検索は英数の大文字小文字を区別しません（内部で小文字化して比較しています）。
* 追従モードにするとフレームをドラッグできなくなります。位置を直したいときは追従を切ってください。
