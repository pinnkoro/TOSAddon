# Boss Direction

> ボスが向いている方向を矢印でお知らせ

| 項目 | 内容 |
| --- | --- |
| キー | `boss_direction` |
| ソース | [boss_direction.lua](boss_direction.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 使い方

ON にするだけです。**視界内（半径 500）に居るボス**の足元に赤い矢印が重なって表示され、
ボスが今どちらを向いているかが分かります。正面攻撃を避ける／背面を取るのに使えます。

ボスの HP が 0 になると矢印は消え、ボスが視界から外れると矢印フレームごと破棄されます。

### レイド「レダニア」での色分け

マップ名に `Raid_Redania` を含むとき、**クラス名に `ILLUSION` を含まないボス**の矢印だけ
**黄色**になります。本体と幻影を矢印の色で見分けるためのものです。

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから設定画面を開きます。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| フレームレイヤー設定 | `29` | 矢印フレームの重ね順。他の UI に隠れる／かぶるときに調整する。**Enter キーで確定** |

## しくみ

0.5 秒間隔のタイマーで次を繰り返します。

1. `SelectObject(自分, 500, "ENEMY")` で敵を走査し、`info.GetTargetInfo(handle).isBoss == 1` の
   ものだけを対象にする。ただし**アイコンが `icon_item_nothing` のモンスターは除外**する。
2. 対象ごとに `boss_direction_<ハンドル>` フレームを作り、`class_tree_arrow` 画像を
   赤（`FFFF0000`）に着色して置く。
3. `info.GetAngle(handle) - 23` を矢印の角度にする（`-23` は画像の向きの補正）。
4. `FRAME_AUTO_POS_TO_OBJ` でボスに追従させ、HP が 0 なら非表示にする。
5. 今回見つからなかったハンドルのフレームは破棄する。

OFF にするとタイマー自体を停止します。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/boss_direction.json`

```json
{ "layer": 29 }
```

## 注意

* **街（`City`）では動きません。**
* 表示対象は**視界内（半径 500）**のボスのみです。
* 本家 Nexus Addons など、同名の旧初期化関数（`BOSS_DIRECTION_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
