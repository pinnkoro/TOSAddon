# Market Voucher

> マーケットでの取引履歴表示

| 項目 | 内容 |
| --- | --- |
| キー | `market_voucher` |
| ソース | [market_voucher.lua](market_voucher.lua) |
| 設定画面 | なし（マーケットの受取箱に「売上伝票」ボタンが追加されます） |

## 使い方

ON にすると、マーケットでの**売却と購入が自動で記録**されます。

記録を見るには、マーケットの**受取箱（`market_cabinet`）を開き、「売上伝票」ボタン**を押します。
1 行 1 取引で、新しいものが上に来る順で並びます。

| 行の種類 | 表示 |
| --- | --- |
| 販売 | `販売日時 : ... ・ キャラ名 ・ アイテム ・ 数量 ・ 単価 ・ 合計金額 ・ 販売` |
| 購入 | 金色の文字。合計金額の前に `△` が付く |

下部に **売上合計**（販売の合計 − 購入の合計。百万単位の概算も併記）と、
**集計期間**（最古〜最新の日付）が出ます。

* ウィンドウは背景をクリックするか右上の × で閉じます。
  受取箱を閉じると自動的に閉じます。
* **「ログ削除」ボタン**で伝票の内容を消せます。消えるのは `market_voucher.json` の内容だけで、
  テキストログ（下記）はそのまま残ります。

## 記録されるタイミング

| フック | 記録内容 |
| --- | --- |
| `CABINET_GET_ALL_LIST` | 受取箱の**一括受取**。`market_sell` 由来のアイテムをすべて「販売」として記録 |
| `_CABINET_ITEM_BUY` | 受取箱から**個別に受け取った**とき。「販売」として記録 |
| `_BUY_MARKET_ITEM` | マーケットでの**購入**。所持金・取引上限のチェックを通ったあとに記録 |

いずれも記録してから本来の処理（受取／購入リクエスト）を実行します。

## 保存先

| ファイル | 内容 |
| --- | --- |
| `../addons/_nexus_addons_p/<アカウントID>/market_voucher.json` | 伝票に表示する取引の配列。「ログ削除」で空になる |
| `../addons/_nexus_addons_p/<アカウントID>/market_voucher_log.txt` | 追記専用のテキストログ。**削除されない** |

1 行の書式は次のとおりです。

```
YYYY-MM-DD HH:MM:SS/キャラ名/アイテム名/数量/単価/合計金額/sell|buy
```

アイテム名に含まれる `-` は `?` に置き換えて保存し、表示時に戻します
（区切り文字と日付のハイフンを取り違えないようにするため）。

初回のみ、旧 `market_voucher` アドオンの
`../addons/market_voucher/<アカウントID>/settings_2507.json` と
`../addons/market_voucher/log_2507.txt` から引き継ぎます。

## 注意

* **販売の「単価」は手数料を戻した値**を表示します（記録されている合計を 0.9 で割った値）。
  一方、合計金額は**受け取った額そのまま**です。
* 集計されるのは**このアドオンを ON にして以降の取引だけ**です。過去の履歴は取得できません。
* テキストログは追記のみで、自動削除や上限はありません。長く使うと大きくなります。
* 本家 Nexus Addons など、同名の旧初期化関数（`MARKET_VOUCHER_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
