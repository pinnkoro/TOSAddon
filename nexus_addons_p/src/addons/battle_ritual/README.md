# Battle Ritual

> レイド前の儀式を自動化します（ソロの時しか動きません・ミニマップの左上のアイコンで切替）

| 項目 | 内容 |
| --- | --- |
| キー | `battle_ritual` |
| ソース | [battle_ritual.lua](battle_ritual.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 何をするアドオンか

レイドやチャレンジに入るたびに手で押している**自己バフのスキル**を、
入場時にまとめて自動使用します。「儀式」＝毎回同じ順番でバフを撒く行為のことです。

## 使い方

1. アドオン一覧の歯車ボタンから設定画面を開き、**使いたいスキルを登録**する。
2. ON にすると、画面上部（ミニマップの左上あたり）に**アイコン**が出ます。
   **右クリックで自動実行の ON/OFF**（白＝ON、灰色＝OFF）。
3. インスタンスマップ／未知の聖域 3F（`11244`）／ヴェルニケ（`8022`）に入ると、
   登録したスキルを**優先度順に 0.1 秒間隔で自動使用**します。

アイコンはドラッグで移動でき、位置は保存されます（設定で固定も可）。

### スキルを飛ばす条件

* **クールタイム中**のスキルは使いません
* **対応するバフ ID を指定していて、そのバフが既に付いている**場合は使いません
  （バフ ID を空欄にすると毎回かけ直します）
* クイックスロットに**入っていないスキルは使えません**

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから開きます。
1 行につき「スキル・バフ・優先度」を設定します。

| 列 | 内容 |
| --- | --- |
| `Add`（左） | スキル一覧を開いて選ぶ |
| スキル ID 入力欄 | 直接入力もできる（錬成スキルは `141001` 以降） |
| `×`（左） | その行を削除 |
| `Add`（中央） | バフ一覧を開いて選ぶ |
| バフ ID 入力欄 | 直接入力。**対応バフが無い場合は空欄**（例: テンペストショットは空欄、ダブルアタックは `322`） |
| `×`（中央） | バフの登録だけ解除 |
| 優先度 | 数字が小さいほど先に使う |
| フレーム固定 | アイコンのドラッグ移動を禁止する |
| フレーム初期位置 | アイコンの保存位置を破棄する |

一番下には常に空行があり、そこから新しいスキルを追加します。

## オーバーロード（レリックバフ）の自動解除

コンテンツ記録の要求（`REQ_PLAYER_CONTENTS_RECORD`）が飛んだとき、
**バフ `40049` が付いていればスキル `100085` を自動で使用**します。
レイド開始時にレリックバフを切るための処理です。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/battle_ritual.json`

```json
{
  "skills": { "<スキルID>": { "buff_id": 0, "priority": 1 } },
  "etc": { "x": 0, "y": 0, "move": 0, "use": 0 }
}
```

`etc.use` が自動実行の ON/OFF、`etc.move` が `1` でフレーム固定です。

## 注意

* **ソロのときしか動きません。**パーティーメンバーが 2 人以上いると何もしません。
* **対象マップはインスタンス／`11244`／`8022`** です。それ以外では自動実行しません。
* スキルは**クイックスロットのアイコン経由**で使用します。
  クイックスロットから外すと動かなくなります。
* 処理全体が `pcall` で包まれているので、エラーが出てもゲームは落ちませんが、
  その場合は静かに停止します。
