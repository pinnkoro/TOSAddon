# Characters Item Serch

> 各キャラクターのアイテム検索（インベントリボタン右クリックでも作動）

| 項目 | 内容 |
| --- | --- |
| キー | `characters_item_serch` |
| ソース | [characters_item_serch.lua](characters_item_serch.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタンで検索画面が開きます） |

## 何をするアドオンか

「あのアイテム、どのキャラが持っていたっけ」を解決するアドオンです。
**インベントリ・装備・個人倉庫・チーム倉庫の内容をファイルに記録**しておき、
ログインしていないキャラのぶんも横断で検索できるようにします。

## 使い方

**メニューバーのインベントリボタンを右クリック**すると検索画面が開きます
（アドオン一覧の歯車ボタンからも開きます）。もう一度押すと閉じます。

| 操作 | 動作 |
| --- | --- |
| `キャラクター切替` ボタン | 表示するキャラを選ぶ（バラック順に並びます） |
| 検索欄に入力して **Enter** | **全キャラ横断でアイテム名を検索**する |

### 表示モード

**キャラ表示**（既定）では、選んだキャラの持ち物がカテゴリ別のツリーで並びます。

| カテゴリ | 内容 |
| --- | --- |
| （先頭行） | 所持シルバー |
| 装備中 | 装備しているアイテムとモンスターカード |
| 倉庫 | 個人倉庫 |
| 武器 / 防具 / 装飾 / 消耗品 / ジェム / アシスター / レリック / 家具 … | インベントリの中身をマーケット分類で分けたもの |

**検索モード**（検索欄に入力して Enter）では、一致したアイテムを**持っているキャラごと**に
まとめて表示します。チーム倉庫のぶんも対象になり、**チーム倉庫は一覧の最後**に来ます。
検索は大文字小文字を区別しない部分一致です。

## 記録されるタイミング

| きっかけ | 記録される内容 |
| --- | --- |
| インベントリを閉じたとき | インベントリ＋装備＋モンスターカード |
| バラックへ戻る／ログアウト／終了 | 同上 |
| 個人倉庫を閉じたとき | 個人倉庫 |
| チーム倉庫を閉じたとき | チーム倉庫 |

つまり、**各キャラで一度ログインしてインベントリと倉庫を開閉する**と、
そのキャラの内容が検索対象になります。

### 保存を見送る場合

倉庫のアイテム情報を 1 件でも取得できなかったときは、
**その回の保存を中止して前回の記録を残します**。このとき、

```
[CIS] 個人倉庫のアイテム情報を取得できなかったため、今回の保存を中止しました
      検索結果は前回のままです
```

とチャットに表示されます。欠けた内容で上書きして「一部のアイテムが検索から消える」ことを
防ぐための動作です（v1.0.2 でこの通知を追加）。

## 保存先

| ファイル | 内容 |
| --- | --- |
| `characters_item_serch.json` | キャラ名簿（名前・バラックレイヤー・並び順・CID） |
| `characters_item_serch_inventory.dat` | インベントリ |
| `characters_item_serch_equips.dat` | 装備とモンスターカード |
| `characters_item_serch_warehouse.dat` | 個人倉庫 |
| `characters_item_serch_accountwarehouse.dat` | チーム倉庫 |

いずれも `../addons/_nexus_addons_p/<アカウントID>/` の下です。
`.dat` は 1 行 1 アイテムで、`:::` 区切りの

```
<キャラ名>:::<IESID>:::<ClassID>:::<個数>:::<アイテム名>:::<場所>:::<カテゴリ>
```

という形式です（チーム倉庫だけは 1 列目が `チーム倉庫` / `Account Warehouse`）。

初回のみ、旧 `characters_item_serch` アドオンの
`../addons/characters_item_serch/<アカウントID>/` にある
`inventory.dat` / `equips.dat` / `warehouse.dat` から引き継ぎます
（チーム倉庫は旧アドオンに無いので新規です）。

## 注意

* **記録はキャラごとに 1 回はその画面を開かないと作られません。**
  新しいキャラは、ログインしてインベントリを開閉するまで検索に出ません。
* バラック名簿に居なくなったキャラの行は、読み込み時に**検索結果から除外**されます
  （`.dat` の行自体は残ります）。
* 表示はアイコンのみで、強化値やオプションは分かりません。個数だけ表示されます。
* 本家 Nexus Addons など、同名の旧初期化関数（`CHARACTERS_ITEM_SERCH_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、個人倉庫を閉じたときの保存が一度も動いていなかった不具合
  （フック名 `WAREHOUSE_CLOSE` にシングルクォートが混入していた）を修正済みです（v1.0.1）。
