# Quickslot Operate

> クイックスロットの女神ポーションをレイド毎に付け替えます（ストレートモード・保存・読込機能もあります）

| 項目 | 内容 |
| --- | --- |
| キー | `quickslot_operate` |
| ソース | [quickslot_operate.lua](quickslot_operate.lua) |
| 設定画面 | なし（クイックスロット横の `QSO` ボタンを右クリック） |

## 何をするアドオンか

女神ポーション（攻撃用・防御用）は**種族ごとに別アイテム**なので、レイドが変わるたびに
クイックスロットへ入れ直す必要があります。このアドオンは

1. **レイドに入るとき／対象マップに入ったとき、自動で正しい種族のポーションに差し替える**
2. **右 SHIFT で種族を手動で順送りする**
3. **ポーションのスロットにマウスを乗せると種族選択パネルを出す**
4. クイックスロットのレイアウトを**名前を付けて保存・読込**する
5. 3 段のクイックスロットを**まっすぐ揃える**（ストレートモード）

を行います。

対応する種族は **Velnias / Klaida / Paramune / Widling / Forester** の 5 つです。

## 使い方

ON にすると、クイックスロットバーの左に **`QSO` ボタン**が追加されます。

| 操作 | 動作 |
| --- | --- |
| **左クリック** | 保存済みスロットレイアウトの読込メニュー |
| **右クリック** | 設定メニュー |

### 自動差し替え

* **レイド入場ダイアログを開いたとき**（`SHOW_INDUNENTER_DIALOG`）に、そのレイドの
  ダンジョン ID から種族を判定して差し替えます。
* **3 秒間隔でマップを監視**し、対象マップ（レイドマップ 17 種）に入っていれば差し替えます。
* ギルドイベントマップにも対応します。
  * `11285` / `11286` → **Paramune**
  * `11267`（ドラグーン）/ `11257`（バウバス）/ `11290`（アシャーク）→ **Velnias**

差し替え時は**インベントリにあるポーションを優先**します。所持していない場合は
1 種類目の ID をそのままセットします（アイコンだけ置く形になります）。

### 右 SHIFT での順送り

設定メニューで `RSHIFT` を ON にすると、**右 SHIFT を押すたびに**
`Velnias → Klaida → Paramune → Widling → Forester → Velnias …` の順で
クイックスロットのポーションが切り替わります（0.15 秒間隔でキーを監視）。

現在スロットに入っているポーションから今の種族を判定するので、
**1 つでもポーションが入っている必要**があります。

### マウスオーバーでの選択

ポーションが入っているスロットに**マウスを乗せる**と、画面下部に 5 種族の
アイコンパネルが出ます。クリックするとその種族に切り替わります。
パネルは 5 秒で自動的に消えます。

### スロットレイアウトの保存／読込

| メニュー | 動作 |
| --- | --- |
| スロットレイアウト保存 | 名前を入力して、**40 スロットすべての内容**を保存する |
| スロットレイアウト削除 | 保存済みのレイアウトを消す |

保存時に**そのキャラの職業構成も一緒に記録**されるので、読込メニューには
`<セット名> : (職業1, 職業2, …)` の形で並びます。
読込は `QSO` ボタンの左クリックからです。

### ストレートモード

設定メニューの「ストレートモード切替」で、クイックスロット 11〜40 の
横位置を 3 段そろえて表示します（通常は段ごとに少しずらして表示されます）。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/quickslot_operate.json`

```json
{
  "ver": 1.1,
  "slotset": { "<キャラ名>": { "<セット名>": { "jobid_0": 0, "1": { "category": "...", "type": 0, "iesid": "..." } } } },
  "straight": false,
  "rshift": false
}
```

初回のみ、旧 `quickslot_operate` アドオンの
`../addons/quickslot_operate/<アカウントID>/settings_250609.json` から引き継ぎます。

## 注意

* **レイアウトの読込はスロットを丸ごと上書きします。**保存されていない枠は空になります。
* 右 SHIFT のショートカットはゲーム内の他の操作と競合することがあります。既定は OFF です。
* ジョイスティックモードでも動きますが、差し替えの瞬間だけ内部でバーの表示を切り替えます。
* 本家 Nexus Addons など、同名の旧初期化関数（`QUICKSLOT_OPERATE_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、新レイド「ズメイ」に対応済みです（v1.0.0）。
