# No Check

> 各種確認を消します（インベントリ検索窓横にボタンがあります）

| 項目 | 内容 |
| --- | --- |
| キー | `no_check` |
| ソース | [no_check.lua](no_check.lua) |
| 設定画面 | なし（インベントリのボタンから 2 つの機能画面を開きます） |

## 何をするアドオンか

大きく 2 つあります。

1. **確認ダイアログの省略** — 繰り返し作業で毎回出る「本当によろしいですか？」を飛ばす
2. **連続使用フレーム／ゴミ箱フレーム** — 同じ操作を繰り返すための専用ウィンドウ

## 省略される確認

ON にすると、次の確認が出なくなります（ゲーム側の関数を置き換えています）。

| 場面 | 元の確認 |
| --- | --- |
| 欠片アイテムなどの使用 | 使用確認 |
| カードブックの使用 | 使用確認（レジェンドカード枠の開放だけは確認を残します） |
| レジェンドカードの装着 | 装着確認 |
| レジェンド／ゴッデスカードの取り外し | 取り外し確認 |
| エーテルジェムの取り外し | ハイカラージェムと無料抽出対象は確認なしで即実行。それ以外は確認を残す |
| ゴッデス装備・アクセの帰属解除 | 実行確認 |
| チャンネル移動 | 移動確認 |
| アイテム破棄の警告ボックス | **入力欄に必要な文字列を自動で埋める**（`destory_now` と、比較テキストの `[...]` 部分） |
| 金床（`reinforce_131014`）の強化 | 5 段階超えの確認をスキップし、**0.3 秒間隔で連続強化**する |

いずれも、ロック中・条件を満たさない場合などの**安全側のチェックは残しています**
（例: ロック済みアイテムは `MaterialItemIsLock`、既にカードが入っていれば
`AlreadyEquippedThatCardSlot`）。

## インベントリのボタン

ON にすると、インベントリの検索窓の右に**ボタンが 1 つ**追加されます。

| 操作 | 開くもの |
| --- | --- |
| **左クリック** | アイテム連続使用フレーム |
| **右クリック** | ゴミ箱フレーム |

### アイテム連続使用フレーム

1. インベントリのアイテムを**右クリック**して、中央のスロットに登録する。
2. `連続使用` ボタンを押す。

0.5 秒間隔で使い続け、**個数が減らなくなった時点で自動停止**します
（使いきった／使えなくなった、のどちらでも止まります）。
スロットを右クリックすると登録を消せます。

### ゴミ箱フレーム

大量のアイテムをまとめて捨てるためのウィンドウです（7×15 = 105 枠）。

1. インベントリのアイテムを**右クリック**して枠に登録する。登録したものは
   インベントリ側にチェックマークが付く。
2. `スタート` を押す。確認ダイアログで承諾すると、0.5 秒間隔で 1 スタックずつ破棄する。
3. `ストップ` またはウィンドウを閉じると中断する。

捨てるたびに `[<アイテム名>] を [n個] 捨てました` が画面に出て、
すべて終わると `[No Check]End of Operation` と表示されます。
枠を右クリックすると登録を取り消せます。

登録時に弾かれるもの:

* ロック中／一時ロック中のアイテム（`MaterialItemIsLock`）
* 破棄できないアイテム（`ItemIsNotDestroy`。ただし寿命切れのものは捨てられます）
* 製作中・取引中は登録自体を受け付けません

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。

## 注意

* **ゴミ箱フレームは本当にアイテムを破棄します。**確認ダイアログにも
  「リカバリーサービス対象外かも」と出るとおり、戻せません。
  **捨てたくないものは必ずロックしておいてください。**
* 破棄警告ボックスの文字列自動入力は、**確認の意味そのものを無効化します**。
  この機能を使う以上、対象の取り違えは自己責任になります。
* OFF にするとインベントリのボタンを取り除き、確認も元どおり出るようになります。
