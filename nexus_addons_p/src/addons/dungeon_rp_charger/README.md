# Dungeon RP Charger

> 聖域で自動でレリックポイントを補充します

| 項目 | 内容 |
| --- | --- |
| キー | `dungeon_rp_charger` |
| ソース | [dungeon_rp_charger.lua](dungeon_rp_charger.lua) |
| 設定画面 | なし（アドオン一覧の ON/OFF のみ） |
| 原作者 | meldavy さん |

## 使い方

ON にして**未知の聖域 3F（マップ ID `11244`）**へ入るだけです。
以降、レリックポイント（RP）が減ると、インベントリのエクトナイトを使って自動で補充します。

## しくみ

マップ ID が `11244` のときだけ 3.0 秒間隔のタイマーを起動し、毎回次を行います。

1. 現在 RP が **200 以上なら何もしない**（余らせないための下限）。
2. インベントリから補充材を探す。優先順は
   **エクトナイト（`11030451` / `misc_Ectonite_Care`）→ エクトナイト（`11030036` / `misc_Ectonite`、マーケット売却可）**。
   どちらも無ければ何もしない。
3. その材料が**ロック状態なら使わない**。
4. 補充する個数を `floor((最大RP - 現在RP) / 10)` として算出し、所持数を超えないよう丸めてから
   `RELIC_CHARGE_RP` トランザクションを実行する（1 個あたり RP 10 換算）。

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。

## 注意

* **動くのは未知の聖域 3F のみ**です。他のマップでは補充しません。
* ロックしたエクトナイトは消費しません。取り置きたい分はロックしておいてください。
* 使う順番は `misc_Ectonite_Care` が先です。マーケットで売れる方（`misc_Ectonite`）を
  残したい場合、この順序は変更できません。
