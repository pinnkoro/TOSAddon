# Revival Timer

> 普通のタイマー（チャットコマンド `/rtimer` で作動）

| 項目 | 内容 |
| --- | --- |
| キー | `revival_timer` |
| ソース | [revival_timer.lua](revival_timer.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 使い方

設定した秒数を**繰り返しカウントダウン**するタイマーです。
リポップ間隔やギミック周期など、一定間隔で来るものを数えるのに使います。

### 表示・非表示

| 方法 | 説明 |
| --- | --- |
| チャットマクロ `/rtimer` | マクロに `/rtimer` を含めておき、そのマクロを実行するとトグル |
| 右 ALT / 左 ALT | 設定でショートカットを有効にした場合のみ。押すたびにトグル |

表示すると 0 まで数え、0 になったら**自動でもう一度最初から**始まります（無限ループ）。
フレームはドラッグで移動でき、位置は保存されます。

### お知らせ

残り **10 秒**と **5 秒**のタイミングで、設定した「お知らせメッセージ」を通知します。

| 設定 | 出力先 |
| --- | --- |
| PT チャット表示 | パーティーチャットへ `<メッセージ> 10秒前` を送信 |
| ニコチャット表示 | 画面を右から左へ流れるニコニコ風テキストで表示（自分だけ見える） |

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから開きます。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| お知らせメッセージ | 空 | タイマー上部に赤字で出る文字列。10 秒前／5 秒前の通知にも使われる。**Enter キーで確定** |
| 秒数設定 | `60` | カウントダウンする秒数。**Enter キーで確定** |
| PT チャット表示 | OFF | 10 秒前／5 秒前をパーティーチャットへ流す |
| ニコチャット表示 | OFF | 10 秒前／5 秒前を画面に流す |
| ショートカット使用 (Right ALT) | OFF | 右 ALT で表示をトグル |
| ショートカット使用 (Left ALT) | OFF | 左 ALT で表示をトグル |
| テスト表示 | — | タイマーを 10 秒だけ表示して動きを確認する |

## しくみ

* `/rtimer` は `EXEC_CHATMACRO` を**置き換えて**実装しています。チャットマクロの本文に
  `/rtimer` が含まれていればタイマーをトグルし、それ以外はポーズ実行とチャット送信を
  元の挙動どおり行います。
* ショートカットは 0.1 秒間隔のキー監視で、**200 ミリ秒のクールダウン**を挟んで
  押しっぱなしの連続トグルを防いでいます。
* 通知はカウントごとに 1 回だけ出ます（`10 秒前 → 5 秒前` の順に状態を進め、
  ループの先頭で状態を戻します）。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/revival_timer.json`

```json
{ "x": 400, "y": 300, "set_second": 60, "set_text": "",
  "ptchat": false, "nicochat": false, "shortcut": false, "shortcut_l": false }
```

## 注意

* **タイマーは止まりません。**0 になると自動で次の周回に入ります。止めるには非表示にしてください。
* ALT キーのショートカットはゲーム内の他の操作と競合することがあります。
  使わない場合は設定を OFF のままにしてください。
* 本家 Nexus Addons など、同名の旧初期化関数（`REVIVAL_TIMER_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
