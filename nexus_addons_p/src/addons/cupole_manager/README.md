# Cupole Manager

> クポル未登録キャラでも自動で呼び出します

| 項目 | 内容 |
| --- | --- |
| キー | `cupole_manager` |
| ソース | [cupole_manager.lua](cupole_manager.lua) |
| 設定画面 | なし（クポル管理画面に「デフォルト変更」ボタンが追加されます） |

## 使い方

1. ON にする。
2. **メインキャラでクポルを 3 体セットし、クポル管理画面を閉じる。**
   閉じた時点でそのセットが記録され、まだデフォルトが空なら同じ内容がデフォルトにもなります。
3. 以降、**クポル未登録のキャラで街に入る**と、記録済みのセットが自動で召喚されます。

デフォルトを変えたいときは、クポル管理画面（`cupole_item`）を開いて右下に追加される
**「デフォルト変更」ボタン**を押してください。現在セットしている 3 体がデフォルトになります。
3 体揃っていない状態で押すと `クポルが3体登録されていません` と出て、何も保存しません。

## しくみ

* 記録は**キャラクター（CID）単位**と、キャラをまたぐ **`default`** の 2 系統です。
* **クポル管理画面を閉じたとき**（`CLOSE_CUPOLE_ITEM`）に、装備中のクポル 3 枠を
  `{ id, name }` で保存します。`default` 側は**まだ空の枠だけ**埋めるので、
  既存のデフォルトが勝手に上書きされることはありません。
* 街（`City`）に入ったとき、装備中のクポルに 1 つでも空き（`-1`）があれば自動召喚を試みます。
  適用するセットの優先順は次のとおりです。
  1. **そのキャラの記録**（3 枠すべて埋まっている場合のみ）
  2. **`default`**（3 枠すべて埋まっている場合のみ）。
     そのキャラに何らかの記録があった場合は `デフォルトのクポルセットを適用します` と通知します。
  3. どちらも揃っていなければ `デフォルトのクポルセット未登録` と出して何もしません。
* 召喚は 1.0 秒間隔の更新スクリプトで 1 体ずつ、3 体まで行います。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/cupole_manager.json`

```json
{
  "default":  { "1": { "id": "...", "name": "..." }, "2": {...}, "3": {...} },
  "<CID>":    { "1": {...}, "2": {...}, "3": {...} }
}
```

初回のみ、旧 `cupole_manager` アドオンの
`../addons/cupole_manager/<アカウントID>/settings.json` からキャラ別の記録を引き継ぎます
（キーが数値かつ 4 文字以上のものだけを CID とみなして拾います）。

## 注意

* **街（`City`）でしか動きません。**また、クポルが 3 体とも埋まっているキャラでは何もしません。
* 本家 Nexus Addons など、同名の旧初期化関数（`CUPOLE_MANAGER_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* デフォルトの登録には**クポル 3 体すべて**が必要です。2 体以下では保存されません。
