# Aethergem Manager

> エーテルジェムの付け替えを自動化（キャラ毎の設定が必要です）

| 項目 | 内容 |
| --- | --- |
| キー | `aethergem_manager` |
| ソース | [aethergem_manager.lua](aethergem_manager.lua) |
| 設定画面 | なし（インベントリのボタンを右クリック） |

## 使い方

ON にするとインベントリに**エーテルジェムのアイコンボタン**が追加されます。

| 操作 | 動作 |
| --- | --- |
| **右クリック** | 設定画面（6 セット × 4 枠）を開く |
| **左クリック** | **付け替えを実行**（装着済みなら取り外し、未装着なら装着） |

### 1. セットを登録する（右クリック）

設定画面には 6 行のセットが並びます。各行に 4 枠のスロットがあり、
そこにエーテルジェムを登録します。

| 操作 | 動作 |
| --- | --- |
| インベントリからジェムを**ドラッグ＆ドロップ** | その枠に登録 |
| インベントリのジェムを**右クリック** | 空いている最初の枠に自動で登録 |
| 枠を**右クリック** | その枠の登録を消す |
| 行の左の `use` / `not use` ボタン | **そのキャラで使うセット**を 1 つ選ぶ |

登録したジェムはレベル（`460` / `480` / `500` / `520` / `540`）が枠に表示され、
枠の縁もレベルに応じた色になります。

### 2. 付け替えを実行する（左クリック）

**メイン・サブ合わせて 4 つの武器すべてにエーテルジェムソケットが開いている**状態で押します。

* 4 枠のいずれかにジェムが入っていれば → **取り外し**モード
* どこにも入っていなければ → **装着**モード

装着モードでは、選択中のセットに登録した ClassID と一致するジェムをインベントリから拾い、
**レベルの高い順**に 4 個使います。

処理中は女神装備マネージャー（`goddess_equip_manager`）が自動で開き、
武器の脱着 → ソケット操作 → 再装備を 0.1 秒間隔で順に進めます。
終わると `[AGM]End of Operation` と表示してウィンドウを閉じます。

### エラーメッセージ

| メッセージ | 原因 |
| --- | --- |
| `エーテルジェムソケットが開いた武器を4ケ所装備して使用してください` | ソケット付きの武器が 4 つ揃っていない |
| `インベントリにエーテルジェムが4個ありません` | 選択中のセットに一致するジェムが 4 個未満 |
| `エーテルジェムソケットが空いていません` | 装着しようとした枠にすでに何か入っている |
| `CanUseAfterTutorialClear` | チュートリアル未完了 |

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/aethergem_manager.json`

```json
{
  "set": { "1": { "1": <ClassID>, "2": 0, "3": 0, "4": 0 }, ... "6": {...} },
  "<CID>": { "use_index": "1" }
}
```

`set` はアカウント共通（6 セット × 4 枠）で、**どのセットを使うかだけがキャラ別**です。
`use_index` が `"-1"` のときは未選択です。

初回のみ、旧 `aethergem_mgr` アドオンの
`../addons/aethergem_mgr/<アカウントID>.json` から引き継ぎます。

## 注意

* **インベントリを閉じると処理が止まります。**実行中は開いたままにしてください。
* 処理中は武器スロット（メイン／サブ）が自動で切り替わります。終了時にメインへ戻します。
* 派生版では、`settings.set` が nil のときに落ちる不具合に nil ガードを追加済みです（v1.0.0）。
* 本家 Nexus Addons など、同名の旧初期化関数（`AETHERGEM_MGR_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
