# Sub Slotset

> クイックスロットを追加します

| 項目 | 内容 |
| --- | --- |
| キー | `sub_slotset` |
| ソース | [sub_slotset.lua](sub_slotset.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン＝新規作成、各スロットセットの右クリック＝個別設定） |
| 原作者 | weizlogy さん |

## 何をするアドオンか

**好きな大きさ・好きな位置に、追加のクイックスロットを何枚でも作れる**アドオンです。
スロットにはアイテムやスキルだけでなく、**ポーズ・アビリティ・エモーティコン・
クエストワープ**まで登録できます。

## 使い方

### 作る

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンで設定画面を開き、
値を決めて `Make` を押します。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| Column | `3` | 横のスロット数（**最大 10**） |
| Row | `3` | 縦のスロット数（**最大 10**） |
| Slot Size | `48` | スロット 1 個の大きさ。**画面高さ ÷ 行数**を超えると弾かれます |
| Layer | `90` | 重ね順 |
| キャラクター専用 | OFF | ON にすると**そのキャラだけ**のスロットセットになる（あとから切替可） |

### 使う

| 操作 | 動作 |
| --- | --- |
| アイコンを**ドラッグ＆ドロップ** | スロットに登録 |
| スロット内で**ドラッグ＆ドロップ** | 同じスロットセット内で**入れ替え** |
| スロットを**右クリック** | 登録したものを**使用**（アイテム使用・スキル発動・ポーズ・エモート送信・クエストワープ） |
| スロットを **左 SHIFT + 左クリック** | そのスロットの登録を**削除** |
| 枠を**ドラッグ** | 移動（位置は保存されます） |
| 右上の鍵アイコン付近を**右クリック** | 個別の設定メニュー |

個別の設定メニュー:

| 項目 | 動作 |
| --- | --- |
| スロットセット削除 | 確認のうえ、そのセットを消す |
| スロットセットロック切替 | ロックすると移動・入れ替え・登録ができなくなる |
| スロットセット設定変更 | 行・列・大きさ・レイヤーを変更、共有／キャラ専用も切り替える |

共有のセットには左上に `shared` と表示されます。

## 登録できるもの

| 種類 | 備考 |
| --- | --- |
| アイテム | 個数・強化値・クールタイムを表示。**手持ちに無いときはアイコンが赤くなり `0` 表示** |
| スキル | クールタイムとオーバーヒートのゲージ付き |
| アビリティ | 未習得だとアイコンが赤くなる |
| ポーズ | クリックで実行 |
| エモーティコン | 未所持だと灰色。クリックでチャットに送信 |
| **クエストワープ** | クエストウィンドウのワープアイコンをドラッグして登録。行き先マップ名がスロットに出る。ワープできない状態だとアイコンが赤くなる |

### クエストワープの登録方法

クエストウィンドウで**ワープ可能なクエスト**の行に、このアドオンが小さなアイコンを追加します。
それをスロットへドラッグしてください
（`左クリック: スロットに登録` とツールチップに出ます）。

## エモーティコン画面の改造

エモーティコン画面（`chat_emoticon`）を、**中身の数に合わせて 10 列で並べ直します**。
標準では枠が固定で余白が多いのを詰めるためのものです。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/sub_slotset.json`

```json
{
  "config": { "index": 0 },
  "share":    { "sub_slotset_0": { "etc": { "x":0,"y":0,"row":3,"col":3,"size":48,"layer":90,"lock":false },
                                   "1": { "category": "Item", "clsid": 0, "iesid": "..." } } },
  "personal": { "<CID>": { "sub_slotset_1": { ... } } }
}
```

初回のみ、旧 `sub_slotset` アドオンの
`../addons/sub_slotset/<アカウントID>/settings.json`（共有）と
`../addons/sub_slotset/<CID>.json`（キャラ別）から引き継ぎます。

## 注意

* 行・列は**最大 10**、スロットサイズは**画面高さ ÷ 行数**が上限です。
  超える値を入れると警告して既定値に戻します。
* **アイテムは GUID で覚える**ので、同じアイテムでも別スタックになると赤くなります
  （ClassID での再検索も行うため、通常は自動で復帰します）。
* ロック中のセットは移動も編集もできません。編集したいときは先にロックを外してください。
* 本家 Nexus Addons など、同名の旧初期化関数（`SUB_SLOTSET_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
