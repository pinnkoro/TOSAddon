# Continue Reinforce

> ゴッデス装備連続強化

| 項目 | 内容 |
| --- | --- |
| キー | `continue_reinforce` |
| ソース | [continue_reinforce.lua](continue_reinforce.lua) |
| 設定画面 | なし（女神装備マネージャーの強化タブに操作パネルが追加されます） |

## 使い方

ON にして**女神装備マネージャー →「強化」タブ**を開くと、画面の右下に操作パネルが、
左側のボタン列に `Cancel` ボタンが追加されます。

強化を 1 回実行すると、**成功するか材料が尽きるまで自動で繰り返します**（既定 0.1 秒間隔）。
成功した時点で止まり、対象アイテムの強化値表示を更新します。

`Cancel` ボタンを押すといつでも中断でき、`Continuous Reinforcement Cancelled` と表示されます。

### 操作パネル

| 要素 | 動作 |
| --- | --- |
| 左上の ON/OFF トグル | 連続強化そのものの有効／無効。OFF にすると標準の UI に戻る |
| `clear` | **回数制限を解除**（無制限にする） |
| `normal` | **強化補助剤**を必要数まで自動選択。もう一度押すと解除 |
| `Premium` | **プレミアム強化補助剤**を自動選択。もう一度押すと解除 |
| 数字ボタン（1〜5, 10, 15, 20, 25, 30） | 回数制限をその値に設定 |
| 入力欄 | 回数制限を直接入力（2 桁まで） |
| 下段の ON/OFF トグル | **回数制限に達したとき、制限を自動でクリアする**かどうか |

回数制限を変更すると `強化制限回数を <n> に設定しました` と通知が出ます。
`0`（＝空欄）は無制限です。

### 補助剤の自動選択

`normal` は**レベルの低い補助剤から**必要数を満たすまで選びます。
同レベルの在庫が足りず上位レベルを使った場合は

```
同レベルの補助剤がありません
上位レベルの補助剤を使用します
```

と通知します。補助剤は**強化が成功／中断するたびに自動で選び直します**ので、
連続強化中に手動で入れ直す必要はありません。

### 強化対象の登録

強化タブを開いている間、**インベントリのアイテムを右クリック**すると強化枠に入ります。
武器・防具・装飾はそのまま登録し、それ以外（強化券など）は使用してから登録し直します。

補助剤の所持数が変化すると強化枠をいったん外して入れ直します
（ゲーム側の必要数表示を最新に保つため）。

## 止まる条件

* 強化に**成功した**
* **材料（主材料）が足りなくなった**
* **回数制限に達した**（制限を設定している場合）
* `Cancel` を押した／女神装備マネージャーを閉じた

「回数制限に達したとき制限をクリアする」を ON にしていると、
止まったあと自動的に無制限へ戻ります。

## 保存先

なし。**回数制限・補助剤の選択・ON/OFF はすべてメモリ上のみ**で、
クライアントを落とすと既定値（連続強化 ON・回数無制限）に戻ります。

## 注意

* **本当に強化を繰り返します。**材料は自動で消費されるので、
  回数制限を設定してから使うことを勧めます。
* OFF にすると、ボタンの大きさや位置を含めてゲーム標準の強化 UI に戻します。
* 強化タブ以外（他のタブ）へ切り替えると操作パネルは消えます。
