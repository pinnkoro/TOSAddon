# Another Warehouse (AWH)

> チーム倉庫改造アドオンの保守改造版

| 項目 | 内容 |
| --- | --- |
| キー | `another_warehouse` |
| ソース | [another_warehouse.lua](another_warehouse.lua) |
| 設定画面 | あり（チーム倉庫の歯車ボタン） |
| 原作者 | ebisuke さん |

## 何をするアドオンか

**チーム倉庫の中身を、標準よりずっと見やすい一覧に差し替える**アドオンです。
そのうえで、

* クリックだけでの搬出入（1 個 / 10 個 / 指定数 / 全数）
* シルバーの入出金ボタン（1M / 5M / 10M / 50M / 100M / 全額）
* **倉庫を開いた瞬間に、決めておいたアイテムとシルバーを自動でやりとりする**
* **セットを 10 個まで登録して、まとめて取り出す**

を行います。

## 使い方

ON にしてチーム倉庫を開くと、標準の倉庫リストが差し替わります。
検索欄、設定ボタン、`?`（ヘルプ）、`TAKE SET`、`AW CLOSE`、所持数表示が追加されます。

`AW CLOSE` を押すと、この改造を解除してゲーム標準の倉庫に戻せます。

### 搬出入の操作

| 場所 | 操作 | 動作 |
| --- | --- | --- |
| インベントリ | アイコンを**左クリック** | 1 個 搬入 |
| インベントリ | アイコンを**右クリック** | **全数** 搬入 |
| インベントリ | **左 SHIFT + 左クリック** | 10 個 搬入 |
| インベントリ | **左 SHIFT + 右クリック** | 数量を入力して搬入 |
| チーム倉庫 | アイコンを**左クリック** | 1 個 搬出 |
| チーム倉庫 | アイコンを**右クリック** | **全数** 搬出 |
| チーム倉庫 | **左 SHIFT + 左クリック** | 10 個 搬出 |
| チーム倉庫 | **左 SHIFT + 右クリック** | 数量を入力して搬出 |

同じ内容が `?` ボタンからいつでも確認できます。

### シルバーの入出金

金額入力欄の下にボタンが並びます。

| ボタン | 左クリック | 右クリック |
| --- | --- | --- |
| `C` | 入力を 0 に戻す | 同じ |
| `1M` / `5M` / `10M` / `50M` / `100M` | その額を**足す** | その額を**引く** |
| `←ALL` | 倉庫にあるシルバーを全額入力 | — |
| `→ALL` | 所持しているシルバーを全額入力 | — |

### TAKE SET（まとめ取り出し）

`TAKE SET` ボタンを**左クリック**すると、登録済みセットの一覧が出ます。
選ぶと、そのセットのアイテムを倉庫からまとめて取り出します。
**右クリック**するとセットの編集画面が開きます。

セットは **10 個**まで持て、それぞれに名前を付けられます。
編集画面では

| 操作 | 動作 |
| --- | --- |
| 倉庫のアイテムを右クリック | そのセットに登録 |
| 設定スロットを右クリック | 登録を消す |
| `初期化` | そのセットを空にする（確認あり） |
| `取出し` | そのセットを取り出す |

## 自動搬出入 / 自動入出金

**倉庫を開いた瞬間に自動で動きます。**設定画面（倉庫の歯車ボタン）で有効にします。

| 項目 | 単位 | 説明 |
| --- | --- | --- |
| 自動入出金 | **キャラごと** | 設定した金額になるようシルバーを自動でやりとりする |
| 自動搬出入 | **キャラごと** | 登録したアイテムを自動でやりとりする |
| 自動入出金 金額設定 | 全体 | 目標のシルバー額。**Enter で確定** |
| チーム倉庫の共通設定 | 全体 | **全キャラ共通**で自動搬出入するアイテム |
| チーム倉庫のキャラクター個別設定 | キャラごと | そのキャラだけ自動搬出入するアイテム |

アイテムの登録は設定画面で行います。

| 操作 | 動作 |
| --- | --- |
| インベントリのアイテムを**右クリック** | **チーム共通**の設定に登録 |
| インベントリのアイテムを**左 SHIFT + 右クリック** | **そのキャラ個別**の設定に登録 |
| 設定スロットを**左 SHIFT + 右クリック** | **インベントリに残す個数**を変更 |
| 設定スロットを**右クリック** | 登録を消す |

### ⚠️ トークンが必要です

**自動処理はトークン（バフ `70002`）が無いと動きません。**
無い場合は `[AWH]トークンがないため自動処理を停止します` と表示して止まります
（自動入出金だけは実行されます）。

倉庫情報の取得に失敗したときは 0.1 秒間隔で**最大 5 回まで再試行**し、
それでも駄目なら自動取得をスキップします。

### そのほかのチェックボックス

| 項目 | 説明 |
| --- | --- |
| チェックすると倉庫に 1 個残します | 全数搬出のときに 1 個だけ倉庫に残す |
| チェックすると表示切替 | 一覧の表示形式を切り替える |

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/another_warehouse.lua`

```lua
return {
  ver = 1.1,
  etc = { display_change = 0, leave_item = 0, auto_silver = 0 },
  items = { ... },                       -- チーム共通の自動搬出入アイテム
  take_list = { { name = "Take Items 1", items = {} }, ... },   -- 10 セット
  chars = { ["<CID>"] = { name = "...", money_check = 0, item_check = 0, items = {} } },
}
```

**JSON ではなく Lua 形式**で保存します。`another_warehouse.json` や、旧
`another_warehouse` アドオンの `settings.json` があれば移行します。

## 注意

* **街（`City`）でしか動きません。**
* Character Change Helper と同時には動きません（CCH 側が `[AWH]が作動中です` と出して止まります）。
* 同系統のアドオン（`YAACCOUNTINVENTORY` / `WAREHOUSEMANAGER`）が読み込まれている場合は、
  **それらの初期化関数を無効化**して自分が動きます。
* 本家 Nexus Addons など、同名の旧初期化関数（`ANOTHER_WAREHOUSE_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、設定変更が保存されない不具合を修正済みです（v1.0.0）。
