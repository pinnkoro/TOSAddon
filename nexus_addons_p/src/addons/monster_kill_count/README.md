# Monster Kill Count

> フィールド狩りで倒したモンスターをカウント（アイテムドロップ情報取得）

| 項目 | 内容 |
| --- | --- |
| キー | `monster_kill_count` |
| ソース | [monster_kill_count.lua](monster_kill_count.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン →「マップ情報」一覧） |

## 使い方

ON にしてフィールド／ダンジョンに入ると、画面右上に小さなパネルが出ます。

| 表示 | 内容 |
| --- | --- |
| `Count : n` | **このマップに入ってから倒した数**（`EXP_UPDATE` の回数で数えます） |
| マップ名 | 現在地 |
| `00:00:00` | 滞在時間 |

パネルはドラッグで移動でき、位置は保存されます。

### マップ情報を見る

アドオン一覧（メニューボタン → アドオン一覧）の**歯車ボタン**を押すと、
記録のあるマップが `<マップID> <マップ名>` の一覧で出ます。選ぶと詳細ウィンドウが開きます。

| 表示 | 内容 |
| --- | --- |
| 滞在時間 | そのマップでの通算滞在時間 |
| 討伐数 | 通算の討伐数。右に **`(実績 / 予測 n 体/時)`**（滞在 1 時間以上なら「実績」） |
| 総獲得アイテム数 | 拾ったアイテムの合計個数 |
| アイテム別の行 | `<アイテム名> : n 個` と、`対討伐 %`・`対総数 %`・`秒/個` |

**`Map Reset` ボタン**（確認あり）でそのマップの記録を消せます。
今いるマップをリセットした場合は、メモリ上の集計もその場で 0 に戻ります。

## 記録する条件

次をすべて満たすときだけ記録します。

* マップ種別が **`Field` または `Dungeon`**
* `自分のレベル − マップの推奨レベル ≤ 50`（ただしマップ ID `11244` は常に対象）
* マップ名に `GuildColony_` を**含まない**（ギルド攻城戦マップを除外）
* パーティーメンバーに**チャンネル 11 以上**の相手が居ない

チャレンジモードの討伐数表示が始まると、パネルは消えます。

## 保存先

| ファイル | 内容 |
| --- | --- |
| `monster_kill_count.json` | パネル位置（`frame_x` / `frame_y`）と、記録のあるマップ ID の一覧 |
| `monster_kill_count/<マップID>.json` | マップごとの記録 |

いずれも `../addons/_nexus_addons_p/<アカウントID>/` の下です。

```json
{ "map_name": "...", "stay_time": 0, "kill_count": 0, "get_items": { "<ClassID>": 0 } }
```

マップ記録は**60 秒ごとに自動保存**され、マップを離れたときにも保存されます。

初回のみ、旧 `klcount` アドオンの `../addons/klcount/<アカウントID>/<マップID>.json` から
引き継ぎます（自分のレベル ±50 の範囲のマップだけ）。

## しくみ（一覧が重くならないようにしている点）

「マップ情報」の一覧は、通ったマップが増えるほど読むファイルが増えます。そこで

* **記録があると一度確定したマップ ID は、以降読み直さない**（メモリに覚える）
* **記録が空だと分かったマップ ID は一覧の対象から外す**（`map_ids` から削除）
* ただし**読めなかったファイルは外しません**。ウイルス対策ソフトのロックなどで
  一時的に読めないだけかもしれず、記録を永久に落とすほうが実害が大きいためです

という 2 段の絞り込みを入れています（v1.0.2）。

## 注意

* **街やインスタンスでは記録しません。**
* 討伐数は経験値の更新回数で数えるため、**経験値の入らない敵は数えません**。
* 本家 Nexus Addons など、同名の旧初期化関数（`KLCOUNT_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版での主な修正:
  * 保存フォルダが作られず**討伐数が一切記録されなかった**不具合（v1.0.1）
  * 「マップ情報」を開くと討伐数と滞在時間が **0 に戻ってしまう**不具合（v1.0.1）
  * 新しく通ったマップが一覧に出てこない不具合（v1.0.1）
  * **`Map Reset` が今いるマップで効かなかった**不具合（v1.0.2）
