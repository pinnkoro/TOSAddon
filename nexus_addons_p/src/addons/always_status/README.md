# Always Status

> ステータスを常に表示します

| 項目 | 内容 |
| --- | --- |
| キー | `always_status` |
| ソース | [always_status.lua](always_status.lua) |
| 設定画面 | あり（パネルを右クリック、またはアドオン一覧の歯車ボタン） |

## 使い方

ON にすると、画面に**選んだステータスが並ぶパネル**が常時表示されます。
値は 0.1 秒間隔で更新されます。

| 操作 | 動作 |
| --- | --- |
| パネルを**右クリック** | 表示設定を開く |
| パネルを**ドラッグ** | 移動（位置は保存されます） |
| 左上の `＋` / `−` | **このキャラでの表示／非表示**を切り替える |

## 設定

パネルを右クリック（またはアドオン一覧の歯車ボタン）で開きます。

| 要素 | 動作 |
| --- | --- |
| ドロップリスト | **10 個のセット**から使うものを選ぶ。選択はキャラごとに保存される |
| メモ欄 | 選択中のセットに名前を付ける（**Enter で確定**）。ドロップリストの表示名になる |
| `フレーム初期位置` | パネルの保存位置を破棄する |
| `表示名リセット` | すべての項目の表示名カスタマイズを初期化する（確認あり） |
| 右上のチェックボックス | チェックするとパネルを**固定**（ドラッグ不可）にする |
| 各行のチェックボックス | その項目を**選択中のセットで表示する**かどうか |
| 各行の色見本（11 色） | その項目の**文字色**を選ぶ（全セット共通） |
| 各行の項目名をクリック | **表示名を自由に変更**する（全セット共通） |

セットは 10 個あるので、「攻撃寄り」「防御寄り」「属性確認用」などを作り分けて
キャラごとに使い分けられます。

## 表示できる項目

約 65 項目あり、初期の色は 4 グループに分かれています。

| グループ | 色 | 内容 |
| --- | --- | --- |
| `base` | 緑 | STR / INT / CON / MNA / DEX、装備スコア、アビリティスコア |
| `atk1` | 橙 | 物理・魔法攻撃力、命中、クリティカル、移動速度、キャスト時間比率 など |
| `atk2` | 赤 | 属性別・武器種別・種族別・防具材質別の攻撃力、ボス対象攻撃力、パーフェクト、復讐 |
| `def` | 青 | 防御力、回避、ブロック、各種抵抗、相殺、エリクサー広域 など |

### 値の出し方の特記

| 項目 | 表示 |
| --- | --- |
| STR / INT / CON / MNA / DEX | 基本値 ＋ ステータス強化ぶんの合計 |
| 物理／魔法攻撃力 | `最小~最大`。イベント攻撃力があるときはその値 |
| アビリティスコア | `%` 付き |
| キャスト時間比率 | `%` 付き |
| サイズ別 / 材質別 / 種族別の攻撃・相殺 | **`AllSize_Atk` などの一括値を加算**したうえで、`値 (上限に対する%)` の形で表示（上限は `レベル × 30`） |
| パーフェクト・復讐・各種「極」抵抗・エリクサー広域 | 装備の**ランダムオプション（最大 6 枠）を全装備ぶん合計**し、`_BM` の値も足したもの |

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/always_status.lua`

```lua
return {
  ver = 1.2,
  base  = { frame_X = 0, frame_Y = 0, enable = 1,
            color = { ["PATK"] = "{#FF6600}", ... },
            abbr  = { ["PATK"] = "物理攻撃", ... } },
  chars = { ["<CID>"] = { use_set = 1, on = 1 } },
  ["1"] = { memo = "free memo 1", ["STR"] = 1, ... },  -- セット 1〜10
}
```

**JSON ではなく Lua 形式**で保存します（項目数が多く、読み書きが速いため）。
`always_status.json` や、旧 `always_status` アドオンの
`../addons/always_status/settings.json` があればそこから読み込んで移行します。

## 注意

* 色と表示名は**全セット共通**です。セットごとに変えることはできません。
* 表示・非表示の切り替え（`＋`/`−`）は**キャラごと**、
  使うセットの選択も**キャラごと**、位置と固定はアカウント共通です。
* 項目を増やすほど毎フレームの更新が重くなります。必要なものだけ選んでください。
* 本家 Nexus Addons など、同名の旧初期化関数（`ALWAYS_STATUS_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
