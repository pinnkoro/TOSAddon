# Party Marker

> パーティーメンバーの頭上にアイコンを付けます

| 項目 | 内容 |
| --- | --- |
| キー | `party_marker` |
| ソース | [party_marker.lua](party_marker.lua) |
| 設定画面 | なし（アドオン一覧の ON/OFF のみ） |
| 原作者 | Charbon さん |

## 使い方

ON にするだけです。パーティーを組んでいるとき、**視界内のパーティーメンバーの頭上に
アイコン（`friend_party`）**が表示され、混戦でも味方を見分けられます。

パーティーを抜けた／メンバーが視界から外れた場合、そのアイコンは自動的に消えます。

## しくみ

0.5 秒間隔のタイマーで次を繰り返します。

1. `session.party.GetPartyMemberList(PARTY_NORMAL)` でパーティー名簿を取得。
   **名簿が無い、または自分ひとり（1 人以下）のときは全アイコンを破棄**して終了。
2. 自分を中心に半径 1000 のオブジェクトを走査し、PC かつ名簿に居るハンドルにだけ
   `party_marker<ハンドル>` フレームを作る（レイヤー 80）。
3. `FRAME_AUTO_POS_TO_OBJ` でキャラの頭上に追従させる。
   オフセットは通常 `(-25, -70)`、**マップ名に `GuildColony_` を含む場合のみ `(25, -70)`**
   （攻城戦マップでは他の頭上表示と重なるため、左右を反転してある）。
4. 名簿に居るが視界内に見つからなかったメンバーのフレームは破棄する。

OFF にすると、次のタイマー実行時に全アイコンが破棄されます。

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。

## 注意

* 対象は**通常パーティー（`PARTY_NORMAL`）のみ**です。ギルドメンバーには付きません。
* 表示されるのは**視界内（半径 1000）に居るメンバー**だけです。遠くのメンバーには付きません。
* アイコンは 1 種類のみで、メンバーごとの色分けや変更はできません。
