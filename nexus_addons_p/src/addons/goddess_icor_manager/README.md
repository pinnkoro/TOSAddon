# Goddess Icor Manager

> インベントリにボタンが出来ます（イコルのオプション付与を一覧表示して簡易化します）

| 項目 | 内容 |
| --- | --- |
| キー | `goddess_icor_manager` |
| ソース | [goddess_icor_manager.lua](goddess_icor_manager.lua) |
| 設定画面 | なし（一覧画面の中にディレイ設定があります） |

## 何をするアドオンか

ゲームの刻印（イコルプリセット）画面は 1 ページずつしか見られず、
「どのページにどの部位のオプションが入っているか」を把握しにくい作りです。

このアドオンは

1. **全刻印ページ × 8 部位のオプションを一覧表示**する
2. **現在の装備と一致するページには★、違うページには付替ボタン**を出す
3. **装備 8 部位ぶんのイコル構成をセットとして保存し、まとめて付け替える**（9 セット）
4. 刻印保存（イコルの刻印）を、一覧の横で行える専用 UI にする

を行います。

## 使い方

ON にして街に入ると、インベントリに**イコルのアイコンボタン**が追加されます。
クリックすると全画面の一覧が開きます（ESC で閉じます）。

### 一覧の見方

左側に刻印ページが 5 つずつ並びます（下部の矢印ボタンで 1〜5 ページ / 6 ページ以降を切替）。
各ページの中に、右手・左手・上着・ズボン・手袋・靴・サブ右手・サブ左手の 8 枠が並びます。

| 表示 | 意味 |
| --- | --- |
| ページ名が黄色 | 使用可能 |
| ページ名が赤字 + `使用不可` | 刻印スロット拡張が切れていて使えないページ |
| 枠の見出しに `[540]` / `[520]上級` / `[500]継承` | そのページに刻印したイコルの種類（枠の色も変わる） |
| **★マーク** | **今の装備とオプションが一致している** |
| **`E` ボタン** | 押すと**そのページのオプションをその部位へ付け替える** |

オプションは種別ごとに色分けされます。

| 色 | 種別 |
| --- | --- |
| 緑 | `STAT` |
| 赤 | `ATK` |
| 青 | `DEF` |
| 紫 | `UTIL_ARMOR` |
| 金 | `SPECIAL` |

### 右側のパネル

| 要素 | 内容 |
| --- | --- |
| 8 部位の枠 | **今装備しているイコルのオプション**（オプションがあれば金枠） |
| 下部の一覧 | 対象ステータス 15 種の現在値と、レベルから計算した目安上限 |
| 武器切替ボタン | 表武器と裏武器を入れ替える |
| `刻印保存` ボタン | 刻印（イコルの彫り込み）を行う専用 UI を開く |
| ディレイ入力欄 | 処理の待ち時間（既定 **0.6 秒**）。**Enter で確定** |
| ドロップリスト | 9 個のセットを選ぶ |

### セットの保存と付け替え

1. 一覧を開いた状態で、ドロップリストからセット（`Set 1`〜`Set 9`）を選ぶ。
2. `保存` を押す。**今装備している 8 部位すべてのオプションが刻印ページのどれかと一致していれば**、
   名前を入力して保存できます。
   一致しない部位があると `付替え不可のため保存できません` と出ます。
3. 以降、そのセットを選んで `付替` を押すと、8 部位ぶんを順に付け替えます。
   `削除` でセットを消せます。

付け替え中は

```
[GIM]セット適用中
バグ防止のため操作をしないでください
```

と表示されます。終わると `[GIM]End of Operation`、
時間内に終わらなければ `[GIM]エラーのため動作を終了します` と出ます。

付け替えは **防具 → 表武器 → 武器の入れ替え → 裏武器 → 武器を元に戻す** の
5 段階に分かれており、裏表で持ち替えが必要な構成にも対応します。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/goddess_icor_manager.lua`

```lua
return {
  delay = 0.6,
  ["<CID>"] = {
    drop_items = { { set_name = "Set 1", set = { RH = "3:::RH", ... } }, ... },  -- 9 個
    icor_ids   = { ["3"] = { RH = <イコルのClassID> } },
  },
}
```

**JSON ではなく Lua 形式**で保存します。
初回のみ、旧 `goddess_icor_manager` アドオンの
`../addons/goddess_icor_manager/settings.json` から引き継ぎます
（このとき、`system_option` に記録のあるキャラのぶんだけを拾います）。

## 注意

* **街（`City`）でしかボタンが出ません。**
* 付け替えには**8 部位すべてに装備が必要**です。空きがあると
  `装備を8ヶ所に着けてから起動してください` と出ます。
* **ロック中の装備は付け替えられません**（`MaterialItemIsLock`）。
* 保存したセットの刻印ページが使用不可（拡張切れ）になっていると、
  `保存しているイコルが使用不可のため、終了します` と出て中止します。
* 処理は**ディレイ秒ごとに 1 部位ずつ**進みます。通信が遅い環境ではディレイを長めにしてください。
* 一覧を開くとゲーム側のステータス画面と女神装備マネージャーを内部で開閉します。
  一覧を閉じたときに元へ戻します。
