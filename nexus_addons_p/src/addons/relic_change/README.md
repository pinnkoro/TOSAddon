# Relic Change

> レリックシアンジェム付替えを簡易に

| 項目 | 内容 |
| --- | --- |
| キー | `relic_change` |
| ソース | [relic_change.lua](relic_change.lua) |
| 設定画面 | なし（ボタンの右クリックメニューで位置設定） |

## 使い方

ON にして**街に入る**と、画面右上あたりにレリックのアイコンボタンが出ます。

| 操作 | 動作 |
| --- | --- |
| **左クリック** | シアンジェムの入れ替えを開始 |
| **右クリック** | フレーム設定メニュー（固定 / 位置を戻す） |

左クリックすると、インベントリとレリックマネージャーが自動で開き、
**インベントリにあるレリックシアンジェムの一覧**がコンテキストメニューで出ます。
付け替えたいジェムを選ぶと、

1. 装備中のレリックから今のジェムを外す
2. 1 秒後、選んだジェムを装着する
3. 1 秒後、レリックマネージャーとインベントリを閉じ、`[RC]End of Operation` と表示する

まで自動で進みます。

一覧に出るのは**同じ種類のジェムのうち一番レベルが高い 1 個だけ**です
（`Gem_Relic_Cyan` を含むアイテムを走査し、`GemLevel` が最大のものを採用）。
表示は `<ジェム名>Lv.<レベル>` の形式です。

## 設定（ボタン右クリック）

| 項目 | 説明 |
| --- | --- |
| フレーム固定 | ドラッグ移動の可否を切り替える（既定は移動不可） |
| フレーム位置を戻す | 保存した座標を破棄し、既定位置に戻す |

どちらもフレームを作り直して即座に反映されます。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/relic_change.json`

```json
{ "x": 0, "y": 0, "move": 0 }
```

`x` / `y` がどちらも `0` のときは既定位置（画面右上から下へ 300・右から 50）を使います。

## 注意

* **街（`City`）でしか表示されません。**
* 選んだジェムが**ロック状態**のときは `MaterialItemIsLock` を表示して装着を中止します
  （このときは古いジェムが外れたままになるので、ロックを外して再実行してください）。
* 本家 Nexus Addons など、同名の旧初期化関数（`RELIC_CHANGE_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
