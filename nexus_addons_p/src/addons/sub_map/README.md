# Sub Map

> 小さなマップを表示します

| 項目 | 内容 |
| --- | --- |
| キー | `sub_map` |
| ソース | [sub_map.lua](sub_map.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 何をするアドオンか

**画面に常駐する小さなマップ**を追加します。ゲーム標準のミニマップより情報量が多く、

* パーティー／ギルドメンバーの位置と生死
* ボス（設定次第で雑魚も）の位置
* ワープポイント、宝箱、ヴァカリネ像・ゼミナ像などの状態
* 未探索エリア（赤い霧）

を重ねて表示します。

## 使い方

ON にすると画面右上に出ます。

| 操作 | 動作 |
| --- | --- |
| ドラッグ | 移動（位置は保存されます） |
| 左上の `＋` / `−` | 折りたたみ／展開 |
| マップ上で **左 CTRL + 左クリック** | その地点を**マップリンク**としてチャットへ貼る |
| マップ上で **左 CTRL + 右クリック** | **ミニマップモード**の切り替え |
| マウスを乗せる | 一時的に最前面へ（外すと元のレイヤーに戻る） |

### ミニマップモード

サイズ 310×350 の固定表示になり、右下に**サーバー時刻**が出ます。
攻城戦（`colony_battle_info` が出ているとき）は左下に**残り時間**も表示されます。

## 表示されるもの

| アイコン | 内容 |
| --- | --- |
| 矢印 | 自分の位置と向き |
| 弓アイコン | パーティーメンバー |
| 杖アイコン | ギルドメンバー |
| 倒れたアイコン | HP 0 のメンバー |
| モンスターアイコン | ボス（`MonRank` が `Boss`）。0.5 秒間隔で追従 |
| 宝箱／プレゼント | 宝箱ポイント。**未開封は箱、開封済みは別アイコン**。中身のアイテム名がツールチップに出る |
| 像アイコン | ヴァカリネ像・ゼミナ像・クレイペダ広場像・オルシャの女神像。**未起動は灰色** |
| ポータルアイコン | ワープポイント（`minimap_portal` / `minimap_erosion`） |
| 赤い半透明タイル | **未探索エリア**（霧のあるマップのみ） |

メンバーのアイコンにマウスを乗せると、名前・レベル・HP・SP が出ます。

### チャレンジモード

次のいずれかに当てはまるとき「チャレンジ扱い」になり、
ワープポイントや像などの固定アイコンを描かず、**代わりにパーティーメンバーの追跡表示**を強化します。

* マップ名に `GuildColony_` を含む
* パーティーのチャンネルが 11 以上
* チャレンジモードの討伐数表示が始まった

このとき、モンスター `160063` / `160055` にも専用アイコンが付きます。

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから開きます。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| サイズ設定 | `200` | マップの一辺（**150〜350**）。範囲外は弾かれる。**Enter で確定** |
| チェックするとフレーム固定 | OFF | ドラッグ移動を禁止する |
| チェックするとチャレンジでミニマップモード | OFF | チャレンジ中だけミニマップモードにする |
| チェックするとチャレンジでのみ表示 | OFF | 通常マップでは非表示にする |
| チェックすると地域名表示 | OFF | マップ上に地域名を重ねる |
| チェックするとチャレンジで雑魚を表示 | OFF | ボス以外のモンスターも小さいアイコンで出す |
| フレーム初期位置 | — | 保存位置を破棄する |
| フレームスキン変更 | `無し` | 背景を `無し` / `薄め` / `濃いめ` から選ぶ |

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/sub_map.json`

```json
{ "ver": 1.1, "visible": 1, "x": 0, "y": 0, "skin_name": "None", "move": 1,
  "size": 200, "minimap": 0, "challenge_minimap": 0, "challenge_only": 0,
  "loc_name": 0, "mob_display": 0 }
```

## 注意

* **インスタンスマップでは表示されません。**
* 「チャレンジで雑魚を表示」を ON にすると、敵が多い場所でアイコンが大量に増えて重くなります。
* マップアイコン（宝箱・像）の更新は**1 秒に 1 回まで**に制限しています。
* 本家 Nexus Addons など、同名の旧初期化関数（`SUB_MAP_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
