# Guild Event Warp

> 画面右上の小さいボタンから封鎖戦マップの 1 チャンネルにワープ

| 項目 | 内容 |
| --- | --- |
| キー | `guild_event_warp` |
| ソース | [guild_event_warp.lua](guild_event_warp.lua) |
| 設定画面 | なし（ボタン列の開閉のみ、状態は保存されます） |

## 使い方

ON にすると**画面右上**にモンスターアイコンのボタンが 3 つ並びます。

| アイコン | 行き先 | イベント ID |
| --- | --- | --- |
| ドラグーン | ギルドイベント「ドラグーン」のマップ | 500 |
| アラクネ姉妹 | ギルドイベント「アラクネ姉妹」のマップ | 501 |
| バウバス | ギルドイベント「バウバス」のマップ | 502 |

クリックするとそのマップへ移動し、**移動後に自動でチャンネル 1 へ切り替えます**
（すでに 1ch のときは切り替えません）。

右端の矢印ボタンでボタン列を**折りたたみ／展開**できます。この状態は保存され、
次回ログイン時も維持されます。

## しくみ

* ボタンは `slot` にモンスタークラスの `Icon` を貼っただけのもので、
  クリックで `control.CustomCommand("MOVE_TO_ENTER_NPC", <イベントID>, 1, 0)` を呼びます。
  これはギルド活動 UI（`guild_activity_ui`）の封鎖線ランキング「移動」ボタンと同じ処理です。
* 移動前に**移動可否チェック**（`g.guild_event_warp_can_move`）を通します。
  次のいずれかに当てはまるときは `ThisLocalUseNot` を表示して移動しません。
  * 統合サーバー／PVP フィールド／PVP サーバーに居る
  * レイヤーが 0 以外（レイヤー変更中）
  * マップ種別が `Dungeon`
  * マップの `Keyword` に `IsRaidField` または `WeeklyBossMap` を含む
* 移動が成立するとフラグを立て、移動先マップでの初期化時に
  `RUN_GAMEEXIT_TIMER("Channel", 0)` でチャンネル 1 へ移動します。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/guild_event_warp.json`

```json
{ "open": true }
```

`open` はボタン列を展開しているかどうかです。

## 注意

* 本家 Nexus Addons など、同名の旧初期化関数（`GUILDEVENTWARP_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、旧クライアントの削除済み内部関数を呼んでいて移動できなかった不具合と、
  未知／インスタンスマップで可否チェックがエラーになりワープ機能全体が止まる不具合を
  修正済みです（v1.0.0）。
