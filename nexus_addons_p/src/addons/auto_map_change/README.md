# Auto Map Change

> マップ移動時のダイアログ選択を自動化

| 項目 | 内容 |
| --- | --- |
| キー | `auto_map_change` |
| ソース | [auto_map_change.lua](auto_map_change.lua) |
| 設定画面 | なし（アドオン一覧の ON/OFF のみ） |

## 使い方

ON にするだけです。以降、**推奨レベルより高いマップへ入ろうとしたときの確認ダイアログ**が
自動で「はい」に答えられ、そのままマップ移動します。

高レベル帯のマップを行き来するときに毎回出る確認を消すのが目的で、
それ以外のダイアログ（クエスト会話・NPC の選択肢など）には触りません。

## しくみ

`DIALOG_CHANGE_SELECT`（選択肢付きダイアログが開いたときのイベント）を購読し、
ダイアログ文字列に `HighLvZoneEnterMsgCustom` が含まれるときだけ
`control.DialogItemSelect(1)` → `control.DialogOk()` を実行します。

判定は文字列一致なので、対象はこの 1 種類のダイアログに限定されます。

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。

## 注意

* OFF にすると、次のダイアログから通常どおり確認が出ます（再ログイン不要）。
* 自動で選ぶのは選択肢の 1 番目です。ゲーム側のダイアログ構成が変わると
  意図しない選択肢を選ぶ可能性があります。
