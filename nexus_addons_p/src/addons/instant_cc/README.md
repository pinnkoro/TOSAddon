# Instant CC

> キャラクターチェンジを簡易にします

| 項目 | 内容 |
| --- | --- |
| キー | `instant_cc` |
| ソース | [instant_cc.lua](instant_cc.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |
| 原作者 | ebisuke さん |

## 使い方

ON にすると、バラックへ戻る操作をしたときに
**「Barrack Charactor List」コンテキストメニュー**が開きます。

| 項目 | 動作 |
| --- | --- |
| `Return To Barrack` | 通常どおりバラックへ戻る |
| `Lv<レベル> <キャラ名> (<職業>)` | **そのキャラへ直接キャラチェンジする** |

キャラを選ぶと、バラック画面を経由しつつ

1. `hidelogin`（ログイン画面をスキップ）にチェックを入れる
2. そのキャラが居るバラックレイヤーを選ぶ
3. そのキャラを選択する
4. ゲームへ入る

まで自動で進みます。バラック画面での手動操作は不要です。

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから開きます。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| チェックするとバラックごとに表示 | OFF | ON にすると `Barrack 1` / `2` / `3` が並び、選んだバラックのキャラだけを一覧する |

キャラが多い環境では ON にするとメニューが短くなります。

## しくみ

* バラック画面を開くたびに、そのレイヤーのキャラ情報
  `{ 名前, レイヤー, 並び順, 職業ID, 性別, レベル, CID }` を保存します。
  一覧はレイヤー → 並び順の順にソートされます。
* バラック名簿に居なくなったキャラ（削除済み）は一覧から消します。ただし
  **クライアント起動直後は名簿件数が 0 で返る**ため、その回は消しません
  （全キャラを誤って消さないためのガードです）。
* キャラを選ぶと `barrack.SelectBarrackLayer` → `barrack.SelectCharacterByCID` を実行し、
  0.2 秒間隔で「選択されたキャラが目的の CID か」を確認してから `BARRACK_TO_GAME()` を呼びます。
  一致しない間はレイヤー選択からやり直し、**登録キャラ数を超えて失敗すると**
  ログイン画面へ戻して `キャラクターの自動取得に失敗しました 手動で選択してください` と表示します。
* キャラチェンジ前に `Indun_list_viewer_CHECK_ALERT("Barrack")` を通します
  （掃討バフなど、離脱前に警告すべき状態があるときは中止します）。
* `BARRACK_START_FRAME_OPEN` のフックは `_G["norisan"]["HOOKS"]` に名乗りを上げた
  アドオンだけが行うので、同系統のアドオンと二重にフックすることはありません。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/instant_cc.json`

```json
{
  "characters": { "<キャラ名>": { "name": "...", "layer": 1, "order": 0,
                                  "jobid": 0, "gender": 0, "level": 1, "cid": "..." } },
  "per_barracks": false
}
```

## 注意

* **一覧に出るのは、そのクライアントで一度でもバラック画面を開いたことのあるレイヤーのキャラだけ**です。
  全キャラを並べたい場合は、バラック 1 / 2 / 3 へ一度ずつ入ってください。
* `INSTANTCC_DO_CC` / `INSTANTCC_APPS_TRY_MOVE_BARRACK` をグローバルに公開しており、
  他のアドオン（CC Helper など）から呼ばれます。OFF にするとこれらは `nil` になります。
