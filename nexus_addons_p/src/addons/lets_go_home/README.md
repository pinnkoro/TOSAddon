# Lets Go Home

> ホームタウンのホームチャンネルにワープします

| 項目 | 内容 |
| --- | --- |
| キー | `lets_go_home` |
| ソース | [lets_go_home.lua](lets_go_home.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタン） |

## 使い方

ON にすると画面右上に家のアイコンボタンが出ます。

| 操作 | 動作 |
| --- | --- |
| **左クリック** | ホームへワープ |
| **右クリック** | **今いるマップとチャンネルをホームとして登録** |
| **BackSlash + 右 SHIFT** | 左クリックと同じ（ショートカット。設定で無効化可） |

ボタンには**トークンワープの残りクールダウン**が数字で重なって表示されます
（0 になると消えます）。

### ホームの登録

ボタンを右クリックすると `現在のマップとチャンネルをホームにしますか？` と聞かれます。
登録すると、続けて `レティーシャへ移動は使用しますか？` と聞かれ、
「はい」ならホーム到着後に自動でレティーシャまで移動します。

登録できるのは**街（`City`）だけ**です。それ以外では `このマップは設定できません` と出ます。

### ワープの手順

左クリックすると、次の順に「使える手段」を探します。

1. **すでにホームのマップに居る**場合はマップ移動を飛ばす。
2. **クエストワープ**が使えるならそれを使う（都市ごとに決まったクエストを判定）。

   | 都市 | 判定に使うクエスト |
   | --- | --- |
   | クレイペダ | `91055`（受注可能）/ `72165`（完了可能） |
   | オルシャ | `90170` / `90171`（いずれも完了可能） |
   | フェディミアン | `60400`（受注可能） |

3. クエストが使えない場合、**トークンワープのクールダウンが 0 なら**トークンワープする。
4. それも駄目なら、インベントリの**都市ワープアイテム**を使う。

   | 都市 | アイテム ClassID |
   | --- | --- |
   | クレイペダ | `640073` |
   | オルシャ | `640156` |
   | フェディミアン | `640182` |

5. どれも使えなければ `ワープする方法がありません` と表示して中止します。

マップ到着後、**チャンネルが設定と違えば自動でチャンネル移動**し、
「レティーシア移動設定」が ON ならそのままレティーシャへ移動します。

## 設定

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタンから開きます。

| 項目 | 既定値 | 説明 |
| --- | --- | --- |
| ホームタウン | 未設定 | クレイペダ / オルシャ / フェディミアンから選ぶ |
| チャンネル | `1` | 1〜10 ch |
| ボタン位置固定 | ON | チェックするとドラッグ移動できなくなる |
| ボタン初期位置 | — | 保存した座標を破棄して既定位置に戻す |
| ボタン表示設定 | ON | OFF にするとボタンを消す（ショートカットは使える） |
| ショートカット設定 (BackSlash+RSHIFT) | ON | キーによるワープの有効／無効 |
| レティーシア移動設定 | OFF | ホーム到着後にレティーシャまで自動移動する |

## しくみ

* ショートカットは 0.2 秒間隔のキー監視で、`BACKSLASH` と `RSHIFT` の同時押しを見ています。
  この監視はボタンを非表示にしていても動きます。
* ワープ開始時に `ENABLE_WARP_CHECK` を通します。賞金首状態などで移動できないときは
  `WarpBanBountyHunt` を表示して中止します。
* ワープ処理は「1 回のクリックで完結しない」（マップ移動 → チャンネル移動 → レティーシャ移動）ため、
  状態フラグを持っており、**移動先マップでの初期化時に続きを再開**します。
* レティーシャ移動には Guild Event Warp と同じ移動可否チェック（統合サーバー／PVP／
  レイヤー変更中／ダンジョン／レイド地域では不可）が入っています。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/lets_go_home.json`

```json
{ "map": "", "ch": 1, "leticia": 0, "x": 0, "y": 0,
  "move": 1, "display": 1, "short_cut": 1 }
```

`x` / `y` がどちらも `0` のときは既定位置を使います。

## 注意

* ホーム未設定のまま実行すると `マップ未設定です` と出ます。先に登録してください。
* ホームを**右クリックで登録し直すと、レティーシア移動設定はいったん OFF に戻ります**
  （登録直後の確認ダイアログで改めて有効にできます）。
* 本家 Nexus Addons など、同名の旧初期化関数（`LETS_GO_HOME_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
