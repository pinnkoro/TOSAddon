# Indun List Viewer

> レイドのクリア状況を一覧表示（Indun Panel の戻るボタン左クリックでも表示）

| 項目 | 内容 |
| --- | --- |
| キー | `indun_list_viewer` |
| ソース | [indun_list_viewer.lua](indun_list_viewer.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタンで一覧が開き、一覧内の歯車で表示設定） |

## 何をするアドオンか

**全キャラのレイド入場回数と掃討回数を 1 画面で見られる**表を出します。
表からそのままレイドへ入場でき、キャラチェンジもできます。

## 使い方

アドオン一覧（メニューボタン → アドオン一覧）の歯車ボタン、または
**Indun Panel の「戻る」ボタンを左クリック**すると開きます。
右上の ✕ か ESC で閉じられます（ESC で閉じるのは一番手前の 1 枚だけで、
他の Nexus Addons P のウィンドウは開いたままです）。

### 表の見方

1 行 1 キャラで、左から職業アイコン・キャラ名・ハード回数・ソロ/自動回数・メモ・表示チェックです。
**ログイン中のキャラは名前がオレンジ**になります。

| 列 | 内容 |
| --- | --- |
| ハード列 `( n )` | ハードモードの**入場回数**。上限（通常 1 回、Spreader / Falouros は 2 回）に達すると**赤字** |
| ソロ/自動列 `( n )/( m )` | `n` = 自動入場の回数（上限に達すると赤字）、`m` = **掃討（スイープ）回数**（1 以上でオレンジ） |
| メモ | キャラごとの自由記入欄。**Enter で保存** |
| 表示チェック | チェックすると、そのキャラのレイド回数を隠す |

### 上部アイコンからの操作

| 操作 | 動作 |
| --- | --- |
| ハード列のアイコンを**左クリック** | そのレイドの**ハードモード入場ダイアログ**を開く |
| ソロ/自動列のアイコンを**左クリック** | **ソロ入場** |
| ソロ/自動列のアイコンを**右クリック** | **自動入場** |
| 職業アイコン／キャラ名を**左クリック** | **キャラチェンジ**（Instant CC が ON のとき） |
| 自分の職業アイコン／名前を**右クリック** | 表示する代表職を選ぶ |
| バラックアイコン | バラックへ戻る |
| 歯車アイコン | 表示するレイドの設定画面 |
| チェック（左） | **スクロールモード**（高さ 545 で打ち切ってスクロールバーを出す） |
| チェック（右） | 「隠す」指定のキャラを一覧から外す |

職業アイコンにマウスを乗せると、そのキャラの**全職業**が出ます（代表職は赤字）。

## 掃討バフのお知らせ

**バラックへ戻る・ログアウト・終了・キャラチェンジ**のとき、
**残り 12 時間以内の掃討バフ**が付いていると、期限切れアイテムの警告ウィンドウに
そのバフを並べて表示します（使い切らずに消えるのを防ぐため）。

この判定は他アドオンからも `Indun_list_viewer_CHECK_ALERT(種別)` で呼ばれており、
Instant CC のキャラチェンジもこれを通ります。

警告ウィンドウで「OK」を押すと、そのまま元の操作（バラック移動やキャラチェンジ）へ進みます。

## 回数のリセット

**毎週月曜 6:00（サーバー時刻）**を過ぎていると、次に街へ入ったときに
全キャラの入場回数を `?` に初期化し、`[ILV]レイドの回数を初期化しました` と表示します。

回数が実際に記録されるのは**そのキャラで街に入ったとき**なので、
ログインしていないキャラは前回の値（またはリセット後の `?`）が表示されます。

## 表示設定（一覧内の歯車ボタン）

レイドごとに「ハード列に出すか」「ソロ/自動列に出すか」をチェックで選べます。
メモ欄の表示切り替えもここにあります。

## 保存先

`../addons/_nexus_addons_p/<アカウントID>/indun_list_viewer.lua`

```lua
return {
  ver = 1.2,
  options = { reset_time = 0, display_mode = "full", hidden = 0 },
  display = { Memo = 1, Zmei_H = 1, Zmei_S = 1, ... },
  chars = { ["<キャラ名>"] = { layer = 1, order = 0, hide = false, memo = "",
                               president_jobid = "", jobid = "/1/2/3",
                               raid_count = { Z_H = "?", Z_A = "?" },
                               auto_clear_count = { Z_S = 0 },
                               cid = "...", pc_name = "..." } },
}
```

**JSON ではなく Lua 形式**で保存します。`indun_list_viewer.json` や、旧
`indun_list_viewer` アドオンの `settings_2510.json` があれば移行します。

## 注意

* **回数の記録は街でしか行われません。**
* 並び順はバラックのレイヤーと並び順です。順番が正しく出ないときは、
  各バラックへ一度ずつログインしてください。
* 本家 Nexus Addons など、同名の旧初期化関数（`INDUN_LIST_VIEWER_ON_INIT`）を持つ
  アドオンが読み込まれている場合は、競合を避けるため初期化をスキップします。
* 派生版では、新レイド「ズメイ」対応、ハードモードを持たないレイドの nil ガード、
  「Hard」ラベルの位置ずれを修正済みです（v1.0.0）。
