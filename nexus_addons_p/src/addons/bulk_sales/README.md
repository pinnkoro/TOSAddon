# Bulk Sales

> 雑貨屋で大量販売出来るスロットセットを提供

| 項目 | 内容 |
| --- | --- |
| キー | `bulk_sales` |
| ソース | [bulk_sales.lua](bulk_sales.lua) |
| 設定画面 | なし（アドオン一覧の ON/OFF のみ） |

## 使い方

ON にして**対象の商店を開く**と、商店ウィンドウの右隣に `Balk Sales` フレームが出ます。

1. **インベントリのアイテムを右クリック**する。
   → **同じ種類のアイテムがインベントリ内から全部まとめて**販売枠に登録され、
   インベントリ側にはチェックマークが付きます。
2. 画面下部の `Sales Amount ▶` に**売却額の合計**が随時表示されます。
3. 取り消したいものは、**販売枠のスロットを右クリック**します（1 スタックずつ解除）。
4. `SELL` ボタン → 確認ダイアログで「はい」を選ぶと、登録した全アイテムを一括で売却します。

売却後・商店を閉じたときは、フレームと登録内容がクリアされます。

### 対象の商店

| 商店 | 内部名 |
| --- | --- |
| クレイペダの雑貨屋 | `Klapeda_Misc` |
| オルシャの雑貨屋 | `Orsha_Misc` |
| フェディミアンの雑貨屋 | `Fedimian_Misc` |
| 証明書コイン商店 | `CertificateCoin_Shop` を名前に含むもの |

## しくみ

* `DIALOG_CLOSE` を購読し、閉じたダイアログ名が上記のいずれかのときにフレームを作ります。
  商店を開き直したときの二重生成を避けるため、**前回クローズから 0.5 秒は無視**します。
* 販売枠は 15 列 × 33 行（= 495 スロット）です。埋まりきると `MaxCount` を表示して打ち切ります。
* 右クリック時に登録から除外されるもの:
  * **ロック中のアイテム**（黙って飛ばします）
  * `IsEnableShopTrade()` が false のもの → `CannoTradeToNPC` を表示
  * ハウジング系（`Housing_Furniture` / `PHousing_Furniture` / `PHousing_Wall` / `PHousing_Carpet`）
    → `Housing_Cant_Sell_This_Item` を表示
* 売却額は `geItemTable.GetSellPrice(...) × 個数` の積算です。
* 売却は `item.AddToSellList(guid, count)` を全件ぶん積んでから `item.SellList()` を 1 回呼びます。

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。
登録内容はフレームを閉じると破棄されます。

## 注意

* **ロックしたアイテムは登録されません。**売りたくないものはロックしておくのが安全です。
* 右クリックで登録されるのは**同じ ClassID のアイテム全部**です。
  「1 スタックだけ売る」には、登録後に不要なスロットを右クリックで外してください。
* 売却額の表示はクライアント側の計算です。実際の取引額と食い違う場合はゲーム側が正です。
