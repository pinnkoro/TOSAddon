# Tavern of Soul

> 簡易検索用データベース

| 項目 | 内容 |
| --- | --- |
| キー | `tavern_of_soul` |
| ソース | [tavern_of_soul.lua](tavern_of_soul.lua) |
| 設定画面 | あり（アドオン一覧の歯車ボタンで検索画面が開きます） |

## 使い方

ON にすると**画面右上に `D` ボタン**が出ます。クリックすると検索画面が開きます
（アドオン一覧の歯車ボタンからも同じ画面が開きます）。

1. 下段の **`Item` / `Buff` / `Skill` / `Monster`** から検索対象を選ぶ（選択中は赤いボタン）。
2. 入力欄に名前の一部を入れて **Enter**。
3. 一致したものが `( ClassID ) 名前 ( ClassName )` の形式で一覧表示されます。

用途は「アイテム ID やクラス名を調べたい」ときの逆引きです。
アドオンの設定を書くときや、不具合報告で ID を添えたいときに使えます。

モードを切り替えると入力欄と結果はクリアされ、画面サイズも初期状態に戻ります。

## しくみ

* 選んだモードの名前で `GetClassList(mode)` を呼び、全クラスを走査します。
* `Name` プロパティを翻訳した文字列に対する**部分一致（大文字小文字を区別する単純検索）**です。
  正規表現ではありません（`string.find(..., 1, true)`）。
* アイコンは `Buff` のみ `GET_BUFF_ICON_NAME`、それ以外は `Icon` プロパティを使います。
  `Skill` はアイコン名に `icon_` が付いていなければ補います。
  アイコンの読み込みに失敗したものは `question_mark` を表示します。
* **`Icon` が `None` または `ui_CreateMonster` のものは一覧から除外**します
  （表示できないダミーが大量に並ぶのを防ぐため）。
* 画面の高さはマップフレームの高さ −150 と 900 の小さい方を上限、160 を下限とします。
  幅は最も長い行に合わせて広がります。

## 保存先

なし（ON/OFF は `../addons/_nexus_addons_p/<アカウントID>/settings.json` に入ります）。

## 注意

* **検索結果の件数に上限がありません。**空欄のまま Enter を押すと全件が並び、
  描画に時間がかかります。必ず何か入力してください。
* 結果はクリックしても何も起きません。表示専用です。
